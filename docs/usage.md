# 使用说明

## 登录

1. 访问 http://localhost:8080
2. 点击"开始使用"
3. 输入教务系统账号密码
4. 输入教务系统地址
5. 点击"登录"

## 获取课程

登录后自动获取课程列表，显示：
- 课程代码
- 课程名称
- 教师
- 上课时间
- 上课地点
- 课程容量
- 已选人数
- 剩余名额
- 课程状态

## 创建任务

1. 选择要抢的课程
2. 选择抢课策略：
   - **定时抢课**: 在指定时间开始抢课
   - **监控抢课**: 持续监控课程名额
   - **混合策略**: 先定时，再监控
3. 设置定时时间（可选）
4. 点击"创建任务"

## 启动任务

1. 在任务列表中找到要启动的任务
2. 点击"启动"按钮
3. 任务开始执行

## 查看状态

- 实时查看任务状态
- 查看抢课结果
- 查看错误信息

## 错误处理

### 网络错误
- 自动重试，指数退避
- 最大重试5次
- 单个请求30秒超时

### 认证错误
- 提示重新登录
- 自动尝试刷新会话

### 课程已满
- 按心愿清单顺序继续抢课
- 继续监控已满课程

## 最佳实践

1. **提前准备**: 提前登录，获取课程列表
2. **合理设置并发**: 根据网络情况设置并发数
3. **选择合适策略**: 根据课程情况选择抢课策略
4. **监控状态**: 实时查看抢课状态
5. **及时处理错误**: 遇到错误及时处理
